
# 📱 Flemzy Phones Pro - Sales Management System

A complete sales management system with admin approval, seller portal, and profit tracking.

## ✨ Features

### 🔐 Authentication System
- Admin login with full control
- Seller registration (requires admin approval)
- Secure session management
- Logout functionality

### 👑 Admin Dashboard
- Approve/reject new sellers
- Add/edit/delete products
- View ALL sales with profit calculations
- Private profit dashboard (hidden from sellers)
- Low stock alerts
- User management

### 👤 Seller Portal
- View available products
- Sell products (auto-updates stock)
- Personal sales history
- CANNOT add/edit products
- CANNOT see profit/cost prices

### 📊 Smart Features
- Real-time stock updates
- Sales tracking
- Profit analytics
- Mobile responsive design
- Search products
- Low stock indicators

## 🚀 Quick Start

### Admin Login
```

Email: flemzy@admin.com
Password: Admin@123

```

### Test Seller Registration
1. Go to `register.html`
2. Fill in details
3. Admin approves/rejects
4. Login as seller

## 📁 File Structure
```

├── index.html      # Login page
├── register.html   # Registration (pending approval)
├── admin.html      # Admin dashboard (full control)
├── seller.html     # Seller portal (limited access)
├── config.js       # Configuration settings
└── README.md       # This file

```

## 🔧 Configuration

Update `config.js` with your settings:
```javascript
const CONFIG = {
    ADMIN: {
        email: 'your-email@admin.com',
        password: 'your-password'
    },
    LOW_STOCK_THRESHOLD: 5
};
```
